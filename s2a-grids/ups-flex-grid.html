UPS Flex Grid

Grid Behavior
• Desktop (≥1280px): 2–6 cards per row via .two-up through .six-up.
• Tablet (768–1279px): Always 2-up. Add .orphan-full to stretch a lone last card full width.
• Mobile (<768px): Single column, fully stacked.

Usage
<div class=”ups-grid three-up”>
<div class=”ups-grid four-up orphan-full”>

Containers
• Standard: max 1440px content area.
• HD: max 1920px content area.
• Hero offset: max 1920px, margin tokens at breakpoints.
• Fluid: no max width (wip — carousels, sliders).
• No container: full width with overflow (wip).

Columns
• 12 cols: 1440+ down to 768px.
• 6 cols: below 768px.

Margins
• Hero offset tokens: --spacing-4xl, --spacing-2xl, --spacing-lg.
• Desktop/tablet: 83.4% auto margins.
• Mobile (767–360): --spacing-lg token.

Breakpoints
• XL (1440+ HD): stress test up to 2560.
• L (1440–1280): standard desktop.
• M (1279–768): tablet landscape.
• S (<767px): mobile / tablet portrait.
